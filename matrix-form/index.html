FAQ

Não. Ele demonstra boas práticas de HTML semântico e validações no cliente — perfeito para portfolio.

Posso publicar no GitHub Pages?

Sim! Basta subir este projeto para um repositório e habilitar o Pages apontando para a branch principal.
